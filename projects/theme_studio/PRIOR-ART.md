# Prior art — deprioritised

We considered evaluating existing Frappe theme apps (Infintrix, dhwani-ris/frappe_desk_theme, Tekton, etc.) before building. **Decision (2026-05-07): skip.** Build our own, internal use only, no public release. Existing apps don't match our 4-axis spec closely enough to be worth the integration cost, and the public-marketplace ROI doesn't justify the maintenance burden.

If this decision is ever revisited, see git history of this file for the full prior-art digest.
